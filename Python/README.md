## 常用代码片段

#### 四舍五入

```python
>>> round(0.5)
0
>>> round(-0.5)
0
>>> round(0.6)
1
>>> round(0.51)
1
>>> round(-0.51)
-1
```

#### 向上向下取整

```python
>>> import math
>>> math.floor(0.6)
0
>>> math.ceil(0.6)
1
>>> math.ceil(-0.6)
0
>>> math.ceil(-1.6)
-1
```

#### 输入输出

```python
>>> # 输入一个整数
>>> n=int(input())
6
>>> n
6

>>> # 输入一个整型列表
>>> l=list(map(int,input().split()))
1 2 3 4 5 6
>>> l
[1, 2, 3, 4, 5, 6]

>>> # 同一行输出
>>> print(' '.join(map(str,l)))
1 2 3 4 5 6
>>> # 也可遍历输出
>>> for i in l:
...     print(i,end=' ')
...
1 2 3 4 5 6
```
